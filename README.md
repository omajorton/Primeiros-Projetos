# 🎮 Jokenpô Interativo em Python

Este é o **meu primeiro projeto de desenvolvimento**! Desenvolvi este jogo clássico de Pedra, Papel e Tesoura para consolidar os fundamentos da lógica de programação e aplicar estruturas de controle de fluxo de forma prática.

O projeto foi construído do zero, focando na experiência de jogo via terminal, tratamento de exceções, validação de entradas e interatividade contínua com o usuário.

---

## ✨ Funcionalidades

- Pedra, Papel e Tesoura contra o computador.
- Escolha aleatória da jogada do computador.
- Sistema de placar (melhor de três).
- Reinício automático de uma nova partida ao final do jogo.
- Validação de entradas numéricas (0, 1 ou 2).
- Tratamento de exceções para impedir que o programa seja interrompido por entradas inválidas.
- Contagem e exibição do placar após cada rodada.
- Animação "JO", "KEN", "PO!!!" utilizando pausas temporizadas.

---

## 🛠️ Tecnologias e Recursos Utilizados

- **Linguagem:** Python 3
- **random (randint):** Geração da jogada aleatória do computador.
- **time (sleep):** Criação das pausas durante a apresentação do jogo.
- **Tuplas:** Armazenamento das opções Pedra, Papel e Tesoura.

## ⚙️ Como Executar

### Pré-requisitos

Ter o **Python 3** instalado.

### Passos

1. Clone ou baixe este repositório.
2. Abra um terminal na pasta do projeto.
3. Execute:

```bash
python minigame\ jokenpo.py
```

> Caso o arquivo principal tenha outro nome, substitua o comando pelo nome correspondente.
